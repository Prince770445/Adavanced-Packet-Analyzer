# Advanced Packet Analyzer (DPI Engine)

This project is a Deep Packet Inspection (DPI) based Packet Analyzer developed in C++.

## Features
- Packet capture analysis using PCAP files
- Ethernet, IP and TCP header parsing
- TLS SNI extraction
- Application detection (YouTube, Facebook etc.)
- Traffic blocking based on rules
- Multi-thread packet processing
